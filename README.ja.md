# textlint-rule-preset-longrun

ロンランでさまざまな記事を執筆してゆくうえでの校正ルールです。

今日現在は、技術ブログ記事用の基本的なプリセットのみがあります。

# Usage proofreading of text

```
npx textlint articles/filename.md
```
